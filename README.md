# Just Fucking Use PXXL

A no-bullshit guide to why you should stop hand-rolling your infrastructure and use a modern deployment platform like PXXL (or Vercel, Render, Railway).

## What This Is

This is a Next.js + Tailwind CSS landing page inspired by the "Just Fucking Use..." meme ecosystem. It's a passionate manifesto about why developers should embrace modern deployment platforms instead of suffering with hand-rolled VPS setups.

## Features

- 🎨 **Pxxl Purple Theme**: Custom gradient backgrounds with animated orbs
- 📱 **Fully Responsive**: Mobile, tablet, and desktop optimized
- ⚡ **Next.js 14**: Latest Next.js with React 18
- 🎯 **Tailwind CSS**: Utility-first CSS framework
- 🖼️ **Image Optimization**: Next.js Image component with proper optimization
- 🔍 **SEO Ready**: Full metadata and Open Graph tags
- ✨ **Smooth Animations**: Hover effects, transitions, and animated background
- 🌙 **Dark Theme**: Matches Pxxl's dark aesthetic

## Tech Stack

- **Framework**: Next.js 14.2.5
- **Styling**: Tailwind CSS 3.4.3
- **Language**: TypeScript
- **Runtime**: Node.js

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/trovhq/justfuckingusepxxl.git
cd justfuckingusepxxl
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Building for Production

```bash
npm run build
npm start
```
